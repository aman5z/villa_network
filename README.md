Villa Network Management
Multi-Compound Villa Network Design (Dubai)

A practical network infrastructure project involving the design, deployment, and monitoring of a cost-effective network solution across two villa compounds.
The project combines real-world networking implementation with custom monitoring dashboard development to simulate a small Network Operations Center (NOC) environment.

📡 Infrastructure Deployment
Co-designed and deployed a network solution across:
3 villa compounds
42 rooms
160+ active users

Key Responsibilities
Configured routers, switches, and wireless access points
Designed stable WiFi coverage across multi-floor villas
Worked within strict budget constraints
Implemented centralized monitoring using open-source tools

Results
✔ Improved network stability
✔ Reduced downtime
✔ Better performance visibility
✔ Increased end-user satisfaction

🖥️ Villa Network Monitoring Dashboard
A custom-built dashboard created to monitor both WAN and LAN network health.

🎯 Objective
Design a lightweight network monitoring panel to simulate real-world IT network operations monitoring.
The dashboard acts as a mini Network Operations Center (NOC) interface.

🌐 WAN Monitoring
External network monitoring includes:
Embedded Fast.com Internet Speed Test
Public network uptime monitoring
External network status tracking
This allows quick detection of ISP issues or internet performance drops.

🏠 LAN Monitoring
Internal monitoring focuses on local infrastructure visibility.

Features
Local LAN speed test service
Internal monitoring dashboard
LAN-only service access via private IP addresses

Examples:
192.168.x.x services
Local monitoring endpoints
Internal performance testing
These services are accessible only inside the villa network.

🎛️ Admin Dashboard UI
The monitoring dashboard includes:
Dark-mode interface
Floating quick-access menu
Real-time clock
Blogger layout override
Embedded monitoring tools
The goal was to build a simple yet functional monitoring console without deploying a heavy backend system.

Dashboard engineering
Overall, the system simulates a small-scale Network Operations Center (NOC).
